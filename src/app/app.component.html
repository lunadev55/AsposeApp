File Visualization App made with Angular, ASP.NET Core and Aspose.

With this app you are able to View a range of different files as HTML, try it out!

- Supported file extensions:

  • TXT (Text)
  • PPTX (Presentation/Slides Microsoft PowerPoint)
  • DOCX (Microsoft Word Documents)
  • PDF (Portable Document Format)
  • XLS, XLSM, XLSX, XLSB... (Range of different excel types, including macros)
This file format is not supported.
